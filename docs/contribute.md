# Contribute
If you'd like to contribute, follow these steps:
1. Fork the repository
2. Click Code button then click Download ZIP
3. Extract ZIP then open the repo folder and run `install.bat`
4. Make changes and commit
> [!NOTE]
> If you make changes at `Main.py` you can test app by go to src folder then open in Terminal and type `python Main.py` then enter before to commit it
5. Push commits you made to GitHub web
6. Make a pull request

After you've followed these steps, please wait until I respond to your pull request
